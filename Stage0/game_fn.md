## Acceptance Criteria

### Scenario: Increase Score

  Given: game is running with no errors

  When: ball hits racket

  Then: increase score

### Scenario: Game Over

  Given: game is running without errors
  
  When: racket misses ball three times in a game
  
  Then: reset the score to zero
